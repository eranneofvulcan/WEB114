Questions

  1. Name a real-life situation where you would need to use an `if` statement.
  2. Login authentication system, if the username and password match, grant access; otherwise, show an error message
  3. Explain the difference between `==` and `===`. Why is strict equality usually better?
  4. `==` will check if the values are equal but will convert types to match. `===` checks that both the value AND type are the same. Strict equality is typically better because it prevents unexpected bugs that would be caused by accidental type conversions
  5. Give an example (in your own words) of when you would use `&&` (AND) vs `||` (OR).
  6. > I would use `&&` to check if someone can board an international flight because they need both a valid passport AND boarding pass. I'd use `||` to give a discount on their flight if they're a airline worker or a member because at least one condition needs to be true
  7. What does the `!` (NOT) operator do? Give an example of a situation you would use it.
  8. `!` changes a boolean to its opposite (i.e., true becomes false and false becomes true). For example, `if (!isLoggedIn)` would run code only when the user is NOT logged in, and then redirect them to the login page
