Reflection Questions

1. Why is textContent safer than innerHTML?

textContent only deals with plain text, making it safer in situations where the input might not be entirely trusted

2. How are HTML elements connected to JavaScript variables using DOM selection?

HTML elements are connected to JavaScript variables by assigning the result of a Document Object Model (DOM) selection method to a variable.

3. What would happen to the text box with id="text" if I added this line of code above it: text.textContent = "Hello!"?

The text inside that element would be replaced with the word "Hello!".
